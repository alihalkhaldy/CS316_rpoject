
# Usage Example

## Step 1: Install Requirements
```bash
pip install -r requirements.txt
```

## Step 2: Run the Script
Execute the Python script:
```bash
python [cs316]_sustainable_agriculture.py
```

## Step 3: Explore Outputs
- View feature importance plots.
- Review model evaluation metrics in the console or output files.

## Sample Output
After running the script, you'll see outputs like:
- **Accuracy Scores:** Model performance metrics.
- **Feature Importance:** Insights into the significance of features.
- **Comparative Analysis:** A summary of the best-performing model.
